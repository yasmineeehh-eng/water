# WaterRPA 安装使用手册

WaterRPA 是一个用于自动点击图片、输入文本、等待、滚动、按快捷键、截图和激活窗口的小工具。

## 推荐方式：直接使用 exe

如果你拿到的是打包好的版本，直接双击 `WaterRPA.exe` 即可运行，不需要安装 Python。

运行时建议：

- 先点击右上角的“检查”，确认图片路径、等待时间、截图目录等都没有问题。
- 再点击“开始运行”，或按 `F1` 开始。
- 需要停止时点击“停止”，或按 `Esc` 停止。
- 程序日志会保存在 `%APPDATA%\WaterRPA Studio\runtime.log`，自动恢复配置保存在同一目录下的 `last_tasks.json`。

## 源码运行方式

如果你拿到的是源码文件夹，请按下面步骤运行。

1. 安装 Python 3.10 或更高版本。
   - 推荐到 Python 官网下载安装包：https://www.python.org/downloads/
   - 安装时勾选 `Add python.exe to PATH`。

2. 打开命令行。
   - 进入 WaterRPA 文件夹。
   - 在文件夹地址栏输入 `cmd`，然后按回车。

3. 安装依赖。

   ```bash
   py -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

   这里推荐使用 `py` 命令。部分 Windows 电脑里的 `python.exe` 可能指向 Microsoft Store 占位程序，导致 `python --version` 没有正常输出。

4. 启动程序。

   ```bash
   py waterRPA_GUI.py
   ```

## 打包 exe

开发者可以用下面的脚本打包：

```powershell
powershell -ExecutionPolicy Bypass -File .\build_exe.ps1
```

打包完成后，文件会生成在：

```text
dist\WaterRPA.exe
```

## 常见问题

**运行前检查提示图片不存在怎么办？**

重新选择图片，或确认 JSON 配置里的图片路径在当前电脑上真实存在。

**点击开始后没有找到图片怎么办？**

先确认截图清晰、缩放比例一致、目标窗口没有被遮挡。程序会在日志里提示持续查找、超时或识别失败的具体原因。

**停止不及时怎么办？**

新版本已经让找图、等待、循环间隔都支持停止检测。按 `Esc` 后通常会在短时间内停止；如果系统本身正在执行一次鼠标/键盘操作，需等待该操作返回。

**程序出错但没有弹出命令行怎么办？**

查看运行日志：

```text
%APPDATA%\WaterRPA Studio\runtime.log
```
