# WaterRPA 傻瓜式安装使用手册

欢迎使用 WaterRPA！这是一个帮你自动操作电脑的工具。
即使你不懂编程，只要按照下面的步骤一步一步来，也能轻松把它跑起来。

---

## 第一步：下载并安装 Python

WaterRPA 是基于 Python 语言编写的，所以电脑里必须先安装 Python。

1. **下载安装包**
   - 访问 Python 官网下载页面：[https://www.python.org/downloads/](https://www.python.org/downloads/)
   - 点击页面上黄色的 **Download Python 3.x.x** 按钮（版本号 3.10 以上均可）。

2. **开始安装（关键步骤！）**
   - 双击下载好的安装包（例如 `python-3.11.0-amd64.exe`）。
   - **【非常重要】** 在安装界面的底部，一定要勾选 **"Add python.exe to PATH"** 这个选项！如果不勾选，后面会运行不了。
   - 点击中间的 **"Install Now"** 开始安装。
   - 等待进度条走完，出现 "Setup was successful" 表示安装成功，点击 "Close" 关闭。

3. **验证是否安装成功**
   - 按下键盘上的 `Win + R` 键（Win键是键盘左下角那个微软徽标）。
   - 在弹出的框里输入 `cmd`，然后按回车。
   - 在黑色的窗口里输入 `python --version` 然后按回车。
   - 如果屏幕上显示了类似 `Python 3.11.0` 的字样，说明安装成功了！
   - 如果显示“不是内部或外部命令”，说明刚才没勾选 PATH，请卸载 Python 重新安装并勾选。

这步实在搞不懂可以在B站搜索Python安装，有其他UP做好的视频可以参考

---

## 第二步：准备项目文件

1. **获取项目**
   - 将朋友发给你的 `waterRPA` 文件夹解压到电脑的任意位置（例如 D盘）。
   - 确保文件夹里有以下文件：
     - `waterRPA_GUI.py` (这是主程序)
     - `requirements.txt` (这是依赖列表)

2. **打开命令行**
   - 进入 `waterRPA` 文件夹。
   - 在文件夹地址栏（就是显示路径的地方）输入 `cmd`，然后按回车。
   - 此时会弹出一个黑色的命令行窗口，路径显示的正是当前文件夹。

---

## 第三步：安装依赖库

我们需要安装一些“配件”让软件能跑起来。

1. **安装库**
   - 在刚才打开的黑色窗口里，复制粘贴下面这行命令，然后按回车：
     ```bash
     pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
     ```
   - *注：后面加的网址是清华大学的镜像源，可以让下载速度飞快。*

2. **等待安装完成**
   - 屏幕上会哗啦哗啦跑很多字，这是正常的。
   - 等到最后出现 `Successfully installed ...` 字样，并且光标停止跳动，说明所有配件都装好了。

---

## 第四步：运行软件

万事俱备，只欠东风！

1. **启动程序**
   - 双击 waterRPA_GUI.py
     

2. **成功！**
   - 如果一切顺利，你应该能看到 WaterRPA 的操作界面弹出来了。
   - 以后每次想用，只需要双击waterRPA_GUI.py即可。

---

## 常见问题解答 (Q&A)

**Q: 运行 `pip install` 时出现红色报错？**
A: 可能是网络问题。请检查网络，或者尝试把命令里的 `-i ...` 那部分去掉再试一次。

**Q: 太麻烦了，有没有更简单的办法？**
A: 有！让发给你软件的朋友直接发给你一个 `.exe` 文件（也就是他用 PyInstaller 打包好的版本）。那样你什么都不用装，双击就能用。
